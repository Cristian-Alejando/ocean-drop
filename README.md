# Proyecto Login - ODS 6 DMI 🌍💧

## Descripción
Este módulo permite que los usuarios ingresen al sistema de gestión de encuestas sobre la calidad y disponibilidad del agua, alineado con el Objetivo de Desarrollo Sostenible 6.
